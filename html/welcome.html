To run the bytecode files produced by this system, you need an OCaml interpreter containing the Graphics library. You can download such a runtime from the "Run" menu of our editor.

Otherwise, you can generate this file using the following command: 

ocamlc -make-runtime -o ocamlgrun graphics.cma
