How to Copy-Paste here

Copy-Paste is disabled by default from Javascript applications, for security reasons (many users copy-paste passwords, and it would be dangerous to let applications read the clipboard for this very reason).
If you want to use Copy-Paste in this editor, you MUST use the Copy/Cut/Paste items available from the navigator Edit menu (usually, right clicking in the editor should show these items in the contextual menu).
